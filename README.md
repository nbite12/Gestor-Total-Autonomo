# Gestor Total Autónomo (LifeOS)

> **Nota:** Este proyecto ha sido creado enteramente por **Eric Munive Garcia** y forma parte de un **portfolio** diseñado para demostrar habilidades técnicas a empresas y reclutadores.

## 🚀 Sobre el proyecto

**Gestor Total Autónomo (LifeOS)** es una aplicación integral de gestión financiera diseñada específicamente para profesionales autónomos. Permite separar y administrar de manera eficiente tanto las finanzas personales como las profesionales en un único lugar.

Entre sus características clave se incluyen:
- 💼 **Área Profesional:** Gestión de ingresos, gastos, IVA, IRPF y cuotas de autónomo.
- 🏠 **Área Personal:** Control de movimientos, presupuestos, categorías y metas de ahorro.
- 🌍 **Visión Global:** Una perspectiva unificada de la situación financiera y proyecciones a futuro.
- ✨ **Asistente de IA Integrado:** Impulsado por Google Gemini para ayudar con la automatización de tareas y la generación de comandos financieros inteligentes.

Este proyecto refleja conocimientos avanzados en el desarrollo Frontend (React, TypeScript), gestión de estados complejos y automatización de datos, garantizando siempre la privacidad y una experiencia de usuario moderna con soporte de modo oscuro/claro y manejo de sesiones.

## 💻 Ejecución Local

**Requisitos previos:** Node.js

Para probar la aplicación en tu entorno local, sigue estos pasos:

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Configurar el Asistente IA:**
   Añade tu clave API de Gemini al archivo `.env.local` usando la variable `GEMINI_API_KEY`:
   ```env
   GEMINI_API_KEY="tu_clave_api_aqui"
   ```

3. **Ejecutar la aplicación:**
   ```bash
   npm run dev
   ```
